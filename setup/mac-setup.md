# Mac Setup

## Goal

Prepare the local machine for AI security and assurance experiments.

## Checklist

- [ ] Confirm macOS version.
- [ ] Confirm Git is installed.
- [ ] Confirm Python 3 is installed.
- [ ] Confirm a code editor is installed.
- [ ] Confirm Homebrew is installed, if needed.
- [ ] Confirm enough disk space is available for local models and datasets.

## Verification Commands

```bash
sw_vers
git --version
python3 --version
df -h
```
